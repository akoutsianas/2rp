# The generalized Fermat equation Ax^2 + By^r = Cz^p and applications

This is the repository associated to the paper *"The Generalized Fermat equation Ax^2 + BY^r=Cz^p and applications"*, by Pedro-José Cazorla García, Angelos Koutsianas and Lucas Villagra-Torcomian, which will soon be available in arxiv.

The relevant file is ***elimination_step.m***, which carries out the elimination step for the equation
	5x^2 + q^{2n} = y^5,
previously studied by Laradji, Mignotte and Tzanakis. In particular, all computations necessary to prove Theorem 7.7 in our article are done through the referred file.

Please report any bugs to any of the authors.
